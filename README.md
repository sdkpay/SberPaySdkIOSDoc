# Введение

SberPay SDK – набор средств для разработки. С ним можно легко и быстро добавить SberPay к способам оплаты в вашем приложении.
Благодаря SDK клиенты смогут в пару кликов платить внутри магазина и без ввода данных карты

## Демо-приложение

Пример интеграции SDK находится по [ссылке](https://github.com/SPay-team/SPayDemo-iOS). Код приложения можно использовать как референс при интеграции
